Getting started in just 4 simple steps

The easiest way to install PyBossa in your computer is to use a Virtual Machine. This method streamlines the process of configuration and installation of libraries in the virtual machine, so when you are done, you can delete it if you want, without polluting your system.

  1. Install VirtualBox: a free virtual machine hypervisor

    Go to VirtualBox Download page and get the version that suits your system. You will find installer for Mac OS X, GNU/Linux and Windows systems.

    Download VirtualBox

  2. Install Vagrant

    Now, we only need another software to handle all the automated set up and configuration of PyBossa in the Virtual Machine: Vagrant. Download the installer that fits your system, install it and go to the next step.

    Install Vagrant

  3. Download and install PyBossa

    The best way to download the latest version and get everything up in minutes is to clone the official repository.

    Note for Windows users: Turn off autocrlf in your git settings.

    Open a terminal and clone the repository (download and install in almost one command) with the following command:

    • git clone --recursive git@github.com:PyBossa/pybossa.git
     Copy to clipboard

  4. Starting PyBossa

    Now that you have everything in place, all you need to do is to run the following command within the pybossa folder:

    • vagrant up
     Copy to clipboard

    This will create a virtual machine for you, install all the dependencies, and make a PyBossa server ready for you.

    Once the process is done, you will only need to start the server. Run the following commands:

    • vagrant ssh
    • cd /vagrant
    • source vagrant_start.sh
     Copy to clipboard

    In your web browser, open a tab with the following URL: http://127.0.0.1:5000 and enjoy!

Need help?

If you need help in setting up your server, please check the official documentation or send us an e-mail to info@pybossa.com.
