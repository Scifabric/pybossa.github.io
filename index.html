Getting started in just a few steps

The easiest way to install PyBossa in your computer is using Juju technology. This method streamlines the process of configuration and installation of libraries in a virtual machine, so when you are done, you can delete it if you want, without polluting your system.

  1. Install VirtualBox: a free virtual machine hypervisor

    Go to VirtualBox Download page and get the version that suits your system. You will find installer for Mac OS X, GNU/Linux and Windows systems.

    Download VirtualBox

  2. Install Vagrant

    Vagrant is needed to handle all the automated set up and configuration of Juju technology in the Virtual Machine: Vagrant. Download the installer that fits your system, install it and go to the next step.

    Download Vagrant

  3. Install PyBossa Juju charm

    Open a terminal and run:

    git clone https://github.com/PyBossa/pybossa-jujucharm.git

    Access the charm:

    cd pybossa-jujucharm

    Launch the virtual machine and access it:

    vagrant up
    vagrant ssh

    Enable Juju:

    juju init
    juju switch local
    juju bootstrap

  4. Deploy PyBossa Juju bundle

    This command will install PostgreSQL and PyBossa and will connect them:

    juju deployer -c bundle.yaml

  5. Export PyBossa port

    In order to have access to the PyBossa server within the virtual machine, you have to open its port.

    Open a terminal and run:

    juju status

    That command will give you the IP where the PyBossa server is listening. Copy it and pass it to the following command:

    sudo natpybossa IP

  6. Testing PyBossa

    In your web browser, open a tab with the following URL: http://127.0.0.1:7000 and enjoy!

Need help?

If you need help in setting up your server, please check the official PyBossa Juju documentation, PyBossa documentation or send us an e-mail to info@pybossa.com.
