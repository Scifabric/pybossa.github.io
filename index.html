Get started

Install a virtual environment

You have to install Vagrant and VirtualBox to run PYBOSSA. Don't worry, they're free!

Set up PYBOSSA with Juju

Juju allows you to easily install PYBOSSA, so you can start your research project as soon as possible.

Test PYBOSSA

You are ready to go! Play around with PYBOSSA a little, then launch the project that will change the world :D
